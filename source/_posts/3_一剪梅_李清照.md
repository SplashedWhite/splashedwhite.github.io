# <center>一剪梅</center>

**<center>[宋]李清照</center>**

<center>红藕香残玉簟秋，轻解罗裳，独上兰舟。</center>
<center>云中谁寄锦书来？雁字回时，月满西楼。</center>
</br>
<center>花自飘零水自流，一种相思，两处闲愁。</center>
<center>此情无计可消除，才下眉头，却上心头。</center>
<!-- more -->

{% note success %}
刚从微蹙的眉间消失，又隐隐缠绕上了心头。
{% endnote %}
